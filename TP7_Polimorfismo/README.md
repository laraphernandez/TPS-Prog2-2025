## Cómo compilar y ejecutar

### Opción 1: Compilar desde línea de comandos

```bash

cd TP7_Java/src/main/java

# Compilar un ejercicio (ejemplo: Ejercicio 1)
javac EjercicioUno/*.java

# Ejecutar
java EjercicioUno.Main
```
